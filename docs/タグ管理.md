# タグ管理

## タグ一覧

`src/lib/tags.ts` の `TAGS` 配列が正規リストの唯一の管理場所。

| タグ | 対象内容の例 |
|---|---|
| `ゲーム` | RPG・アクション・格ゲー・オンラインゲーム等のプレイ記録 |
| `音ゲー` | DDR・ビートマニア・ドラムマニア等 |
| `プロレス` | 観戦記・DVD感想・21CPB関連 |
| `アニメ` | アニメ・マンガ・映画の感想 |
| `日常` | 学校・大学・仕事・生活全般 |
| `サイト運営` | DLB/OBRN/C2G等のHP更新・再編作業 |
| `Web技術` | CSS・JavaScript・ブラウザ挙動等 |
| `音楽` | CD購入・アーティスト感想（m-flo・KICK等） |
| `創作` | オリバト・OBRN・小説執筆 |
| `読書` | ライトノベル・小説の読書感想 |

---

## タグを追加する手順

1. **`src/lib/tags.ts`** の `TAGS` 配列に新しいタグ名を追記する

   ```typescript
   export const TAGS = [
     // ... 既存タグ ...
     '新タグ名',  // ← 追加
   ] as const;
   ```

2. **`scripts/assign-tags.ts`** の `TAG_RULES` に判定キーワードを追加する

   ```typescript
   {
     tag: '新タグ名',
     keywords: ['キーワードA', 'キーワードB'],
   },
   ```

3. スクリプトを実行して既存記事に一括付与する

   ```bash
   # 変更内容を確認（ファイルは変更しない）
   npx tsx scripts/assign-tags.ts --dry-run

   # 実際に書き込む
   npx tsx scripts/assign-tags.ts
   ```

---

## タグを手動で付与・修正する

記事の frontmatter を直接編集する。タグ名は `TAGS` に定義されたもの以外はビルド時にエラーになる（タイポ防止）。

```yaml
---
date: 2000-04-08
tags: [ゲーム, 音ゲー, 日常]
---
```

---

## タグを削除する手順

1. `src/lib/tags.ts` から該当タグを削除する
2. そのタグを持つ記事の frontmatter を一括修正する（削除するか別タグに変更）
3. `scripts/assign-tags.ts` の `TAG_RULES` からも削除する

---

## 未タグ記事を確認する

```bash
# タグなし記事のパスを一覧表示
grep -rl "tags: \[\]" src/content/posts/
```
