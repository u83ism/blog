# 機能ロードマップ

## 実装済み機能

| 機能 | ルート / 実装箇所 |
|------|-----------------|
| 記事一覧（ページネーション） | `/`、`/page/[page]/` |
| 個別記事ページ | `/posts/[...slug]/` |
| 前後記事ナビゲーション | `PostLayout.astro` |
| タグ一覧 | `/tags/` |
| タグ別記事一覧 | `/tags/[tag]/` |
| タグ別記事数ソート（降順） | `collectTags()`（`posts.ts`） |
| 年別アーカイブ | `/archive/`、`/archive/[year]/` |
| **月別アーカイブ** | `/archive/[year]/[month]/` |
| OGP / Twitter Card | `Layout.astro` |
| View Transitions | `Layout.astro`（`<ClientRouter />`） |
| レスポンシブナビゲーション | `Nav.astro` |
| **全文検索** | `Nav.astro` + `SearchDialog.astro`（Pagefind） |
| **RSSフィード** | `/rss.xml`（`@astrojs/rss`） |
| **サイトマップ** | `sitemap-index.xml`（`@astrojs/sitemap`） |
| **関連記事** | `PostLayout.astro`（共通タグ数スコアで最大5件） |
| **読了時間の表示** | `PostLayout.astro`（日本語500字/分） |
| **パンくずリスト** | `Breadcrumb.astro`（JSON-LD付き） |
