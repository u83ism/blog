## Content Collections スキーマ

`src/content/config.ts`:

```typescript
import { defineCollection, z } from 'astro:content';

const posts = defineCollection({
  type: 'content',
  schema: z.object({
    date: z.date(),
    tags: z.array(z.string()).default([]),
    title: z.string().optional(), // 省略時はdateから自動生成
  }),
});

export const collections = { posts };
```

### frontmatterサンプル

```yaml
---
date: 2000-04-08
tags: ["ゲーム", "モンコレ"]
---

初めて雑記を書いてみたんですけど……
```

- `title` は省略可能。省略時は `YYYY年MM月DD日` の形式で自動生成する
- `tags` は省略時に空配列

## ページ仕様

### 記事一覧（`/`）

- 全記事を日付降順で表示
- ページネーション（1ページあたり20件程度）
- サイドバーにタグ一覧・年別アーカイブへのリンク

### 個別記事（`/posts/YYYY/YYYY-MM-DD/`）

- スラッグはファイルパスから自動生成（`posts/2000/2000-04-08`）
- 前後記事へのナビゲーション
- タグの表示

### タグ別一覧（`/tags/[tag]/`）

- タグを持つ記事を日付降順で表示

### 年別アーカイブ（`/archive/[year]/`）

- 指定年の記事を月別にグルーピングして表示

## タグ設計方針

移行時に主要タグを事前設計したうえで付与する。自動付けは行わない。
候補トピック（雑記内容から）：ゲーム、プロレス、DDR・音ゲー、アニメ・マンガ、日常、サイト運営、Web技術
