- **サイト名**: u83 said so.

- `output: 'static'`（SSG）

- `trailingSlash: 'always'`（DLBと統一。`import.meta.env.BASE_URL` が `/blog/` になる）

- `site: 'https://u83ism.github.io'`

### View Transitions

`Layout.astro` で `<ClientRouter />` を使用し、ページ遷移時にAstroのView Transitionsを有効化している。

## Content Collections スキーマ

`src/content.config.ts`（Astro 6形式）：

```typescript
import { defineCollection, z } from 'astro:content';
import { glob } from 'astro/loaders';

const posts = defineCollection({
  loader: glob({ pattern: '**/*.md', base: './src/content/posts' }),
  schema: z.object({
    date: z.date(),
    tags: z.array(z.string()).default([]),
    title: z.string().optional(), // 省略時はdateから自動生成
  }),
});

export const collections = { posts };
```

### frontmatterサンプル

```yaml
---
date: 2000-04-08
tags: ["ゲーム", "モンコレ"]
---

初めて雑記を書いてみたんですけど……
```

- `title` は省略可能。省略時は `YYYY年MM月DD日` の形式で自動生成する
- `tags` は省略時に空配列

## ロジック層（`src/lib/posts.ts`）

記事操作に関する純粋関数をまとめたモジュール。

| 関数名 | 役割 |
|--------|------|
| `formatDateTitle(date)` | `Date` → `YYYY年MM月DD日` 形式の文字列 |
| `formatDateDisplay(date)` | `Date` → `YYYY/MM/DD` 形式の文字列 |
| `resolveTitle(post)` | 記事タイトルを返す（`title` 省略時は日付から生成） |
| `buildPostUrl(base, id)` | 記事 id から個別ページURLを生成（`.md` 拡張子を除去） |
| `sortPostsByDateDesc(posts)` | 記事を日付降順でソート |
| `collectTags(posts)` | 全タグを集計し出現数降順で返す |
| `groupPostsByYear(posts)` | 記事を年ごとにグループ化（降順） |
| `groupPostsByMonth(posts)` | 記事を月ごとにグループ化（降順） |
| `findRelatedPosts(current, posts, limit?)` | 共通タグ数スコアで類似記事を返す（デフォルト上位5件） |
| `estimateReadingTime(body)` | Markdown本文の文字数から読了時間（分）を推定（500字/分） |

## ページ仕様

### 記事一覧（`/`）

- 全記事を日付降順で表示
- ページネーション（1ページあたり20件）
  - 1ページ目は `index.astro` が担当
  - 2ページ目以降は `page/[page].astro` が担当（`/page/2/`、`/page/3/` …）
- サイドバーにタグ一覧・年別アーカイブへのリンク

### 個別記事（`/posts/YYYY/YYYY-MM-DD/`）

- スラッグはファイルパスから自動生成（glob loaderのidから`.md`拡張子を除去）
- 前後記事へのナビゲーション（PostLayout.astro に実装）
- タグの表示
- OGタイプは `article`

### タグ一覧（`/tags/`）

- 全タグを出現数降順で表示
- 各タグに記事件数を表示

### タグ別一覧（`/tags/[tag]/`）

- タグを持つ記事を日付降順で表示
- ヘッダーにタグ名と件数を表示

### アーカイブトップ（`/archive/`）

- 記事が存在する年の一覧を降順で表示
- 各年に記事件数を表示

### 年別アーカイブ（`/archive/[year]/`）

- 指定年の記事を月別にグルーピングして表示
- 月の見出しは `/archive/[year]/[month]/` へのリンク

### 月別アーカイブ（`/archive/[year]/[month]/`）

- 指定年月の記事を日付降順で表示
- サイドバーに年別アーカイブナビを表示

### RSSフィード（`/rss.xml`）

- 全記事を日付降順で配信（`@astrojs/rss`）

## レイアウト・UI仕様

### Layout.astro

全ページ共通のシェル。以下を含む：

- `<head>`: OGP・Twitter Card・canonical URL・デフォルト description
- `<Header>`: サイトタイトル（`transition:persist` で遷移時も維持）
- `<Nav>`: グローバルナビゲーション（currentPath を渡してアクティブ表示）
- サイドバーグリッド: `sidebar` スロットがある場合のみ2カラム（PC: 220px + 1fr）
- `<ClientRouter />`: View Transitions

#### サイドバーのレスポンシブ対応

- 900px以下: グリッド解除、サイドバーを上部帯として表示
- サイドバーなしのページ（個別記事など）では1カラム表示

### Nav.astro

- PC: 横並びのタブ型ナビゲーション
- モバイル（600px以下）: 「メニュー」ボタンで開閉するアコーディオン型
- ナビリンク: HOME / タグ一覧 / アーカイブ / 🔗 ゆうやみポータル（外部リンク）

### PostLayout.astro

- サイドバーなし・中央集中型（max-width: 800px）
- パンくずリスト（`Breadcrumb.astro`）
- ヘッダーに投稿日と読了時間（約X分）を表示
- 関連記事（共通タグ数スコア上位5件、タグなし記事では非表示）
- 前後記事ナビゲーション: 日付が新しい方が「次の記事（→）」、古い方が「前の記事（←）」

## OGP / SNS メタタグ

`Layout.astro` で全ページに出力。

| プロパティ | 内容 |
|---|---|
| `og:type` | `website`（デフォルト）/ `article`（記事ページ） |
| `og:title` | `ページタイトル \| u83 said so.` |
| `og:description` | Props の `description`（デフォルト: サイト説明文） |
| `og:url` | canonical URL |
| `og:site_name` | `u83 said so.` |
| `og:locale` | `ja_JP` |
| `og:image` | Props の `ogImage`（省略時は出力なし） |
| `twitter:card` | 画像あり: `summary_large_image` / なし: `summary` |
