# デザイン仕様

## コンセプト

**「紙と活字の読み物」**

Xでは書けない長文を受け止める場所として、文芸誌・文庫本の読書体験をWebで再現する。
他の5プロジェクト（DLB・C2G・OBRN・Portal・21CPB）との差別化を意識し、独自の佇まいを持たせる。

### 他プロジェクトとの差別化軸

| プロジェクト | デザイン特徴 | このブログとの違い |
|---|---|---|
| DLB | パステル × ゴシック × ライト | 明朝 × クリームで別物 |
| C2G | ダーク × パープル × ゴシック | ライト固定で別物 |
| OBRN | 廃墟 × Material UI × Roboto | 完全に別方向 |
| Portal | 宇宙 × Tailwind | 完全に別方向 |
| 21CPB | 旧来HTML × テーブルレイアウト | 現代的組版で別物 |

## カラーパレット

| 変数名 | 値 | 用途 |
|---|---|---|
| `--bg` | `#f5f0e8`（クリーム） | 背景色 |
| `--white` | `#ffffff` | カード・引用ブロックなど |
| `--text` | `#2c2c2c` | 本文 |
| `--text-muted` | `#888888` | 日付・補足情報 |
| `--accent` | `#c17f5a`（テラコッタ） | リンク・区切り線・強調 |
| `--accent-dark` | `#a0633d` | アクセントのhover状態 |
| `--border` | `#ddd8ce` | 罫線・区切り |

> **設計意図：** 白ではなくクリームの背景が「紙の本を読む」感覚を生む。テラコッタは朱より柔らかく、素焼きの陶器のような親しみやすい暖色。

## タイポグラフィ

### フォントスタック

```css
font-family: "游明朝", "Yu Mincho", "Hiragino Mincho ProN",
             "HiraMinProN-W3", "ヒラギノ明朝 ProN", serif;
```

- Webフォント不使用（ゼロネットワークコスト）
- Windows / macOS / iOS は游明朝・ヒラギノ明朝が適用される
- Androidはシステムのserifフォールバック

### サイズ・行間

| 用途 | font-size | line-height |
|---|---|---|
| 本文 | `17px` | `2.0` |
| 補足・メタ情報 | `13px` | `1.6` |
| 見出し（h2） | `1.3rem` | `1.4` |
| 見出し（h3） | `1.1rem` | `1.4` |

> **設計意図：** 明朝体は字面が小さいため、ゴシック体より一回り大きなサイズ・広い行間を取る。

## レイアウト

### 記事ページ（PostLayout）

- 本文カラムを画面中央に配置（**カラム集中型**）
- 最大幅：`640px`
- サイドバーなし
- 読書に集中できる一本道の構造

```
┌─────────────────────────────┐
│         [ヘッダー]           │
│                             │
│       ┌──────────┐          │
│       │  本文     │          │
│       │  640px   │          │
│       └──────────┘          │
│                             │
│         [フッター]           │
└─────────────────────────────┘
```

### 一覧ページ（Layout）

- 現状のサイドバー構造を維持（タグ・年別アーカイブナビ）
- PC：サイドバー（220px）＋ 本文グリッド
- モバイル（900px以下）：シングルカラム

---

## ヘッダー

**扉ページ的・中央揃え**

- サイト名「u83 said so.」を明朝体で中央に配置
- 「since 2000」をキャプションとして小さく添える
- 装飾：なし（テキストの格で勝負）
- 現在のグラデーションアクセントバーは廃止

```
         u83 said so.
         since 2000
```

---

## 装飾方針

**最小限の線のみ**

- 記事タイトル下・ヘッダー下などの区切りに**テラコッタの細い横罫（1px）**を使用
- 花形装飾・ドロップキャップ・背景画像は使用しない
- 余白と罫線だけでリズムを作る

---

## ダークモード

**非対応（ライトのみ）**

クリーム紙の世界観を一枚岩で守る。
ダーク版はC2Gが担う。
